简介
==
---
该模组加入了更多雪球种类、工具、武器等一系列物品，让你能更愉快地享受打雪仗的乐趣。

~~小心被雪球打死~~

目前只支持mc1.18.2\
需要fabirc运行，fabric-api作为前置mod\
后续会更新更多内容以及版本移植
